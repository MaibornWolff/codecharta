To visualize a map you first need to load it. Use the **folder-open** button (labeled _"Load cc.json files"_) in the top-left of the navigation bar, next to the logo.

![The Load cc.json files button](/assets/images/docs/visualization/user-controls/upload.png)

Clicking it opens your operating system's file picker, which accepts CodeCharta files — a plain `.cc.json` or a gzip-compressed `.cc.json.gz` (the accepted file types are `.json` and `.gz`). These are the files produced by the [CCSH](/docs/analysis/codecharta-shell); see [Getting Started](/docs/overview/getting-started) for how to generate one.

## Loading multiple maps

The picker allows **multiple selection**, so you can load several maps at once. You can also load more maps later — each upload adds to the maps already available.

Once more than one map is loaded you can:

- pick which one is shown with the [map selector](/docs/visualization/user-controls/change-maps), and
- switch to [Compare](/docs/visualization/user-controls/compare) to see the delta between two of them.

:::note
CodeCharta runs entirely in your browser — the files you open never leave your machine. See [Data Privacy](/docs/overview/data-privacy).
:::
