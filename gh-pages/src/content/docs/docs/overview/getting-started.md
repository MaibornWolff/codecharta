## Install & run CodeCharta

After trying out the Web Studio, you may be wondering how to get your own project metrics into CodeCharta. You can generate metrics with our [CCSH (CodeCharta Shell)](/docs/analysis/codecharta-shell) and then import them into the Web Studio. There are three ways to get up and running:

- [Docker](/docs/overview/getting-started/docker) — use the pre-built CodeCharta containers, no local installation required.
- [npm](/docs/overview/getting-started/npm) — install the CCSH globally via npm.
- [Local](/docs/overview/getting-started/local) — build and run CodeCharta from source.
