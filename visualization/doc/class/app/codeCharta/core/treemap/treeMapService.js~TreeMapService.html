    public transformNode(node: Object, heightKey: string, p: number, heightScale: number) source

    Transforms a d3 node to our specific representation. Our specification requires a third axis z. -add z0 and z1 depending on treedepth and padding -add w,l,h depending on data.type and x, y -all we also write all important stuff into the node itself so we do not need node.data anymore (there could be obsolete data from old cc.jsons) -the height of the new z axis needs to be scaled to the w/l of the {d3#treemap}

    Params:

    NameTypeAttributeDescription
    node Object

    d3 node
    heightKey string

    name of the height metric
    p number

    padding for minimal height, padding and folder height
    heightScale number

    scaling factor
