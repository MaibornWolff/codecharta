    public addBuilding(w: number, h: number, l: number, x: number, y: number, z: number, heightDelta: number, node: object, heightKey: string, showLabel: boolean) source

    Adds a building to the scene root. A building is an Object3D with 1 or 2 children cubes. The amount depends on the delta value of the cube.

    Params:

    NameTypeAttributeDescription
    w number

    width
    h number

    height
    l number

    length
    x number

    x position
    y number

    y position
    z number

    z position
    heightDelta number
    node object

    the transformed d3 node
    heightKey string

    the height metric
    showLabel boolean

    true if the building should get a label

    Test:

    • app.codeCharta.codeMap.codeMapService addBuilding should add a group to the root when heightDelta is >0
    • app.codeCharta.codeMap.codeMapService addBuilding should add a group to the root when heightDelta is <0< a="">
    • app.codeCharta.codeMap.codeMapService addBuilding should add a group to the root when heightDelta is =0
    • app.codeCharta.codeMap.codeMapService addBuilding should add a group to the root when heightDelta < h and heightDelta > 0
    • app.codeCharta.codeMap.codeMapService addBuilding should add a group to the root when -heightDelta > h and heightDelta < 0
    • app.codeCharta.codeMap.codeMapService addBuilding with heightDelta > 0 should request two meshes (delta and building)
    • app.codeCharta.codeMap.codeMapService addBuilding with heightDelta < 0 should request two meshes (delta and building)
    • app.codeCharta.codeMap.codeMapService addBuilding with heightDelta = 0 should request one meshes (only building)
