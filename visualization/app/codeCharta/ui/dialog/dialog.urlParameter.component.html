URL Parameters

Parameters

  • file parameter can be used multiple times. Each accepts a file location, which must be reachable through XHR. With multiple file parameters, you can load more than one map.
  • mode parameter can be used to set one of the three different modes:
    • Single will only show the first imported files
    • Delta will show the first two imported files
    • Multiple will show all imported files aggregated together in one map
